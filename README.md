# Fundamentos React Workshop | OpenWebinars 

Este repositorio contiene las prácticas y scripts desarrollados durante el curso de **Fundamentos de React** de **OpenWebinars**. El código está organizado por carpetas que corresponden a los bloques prácticos del programa.

---

## Contenidos del Proyecto

Dentro de la carpeta principal encontrarás las siguientes carpetas, organizadas siguiendo la estructura de módulos del curso para facilitar el seguimiento práctico:

* **01-componentes-y-eventos**: Creación de proyecto, estilos CSS, eventos y reactividad.
* **02-props-y-listas**: Uso de Props, renderizado condicional y manejo de listas.
* **03-hooks-api-y-formularios**: Ciclo de vida, Hooks (State/Effect), formularios y Fetch API.
* **04-proyecto-final**: Aplicación real integrando los conocimientos adquiridos.
